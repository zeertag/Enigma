Правила работы с машиной.

1) Настройки машины
Настройки заполняются либо в файл settings, либо через консоль,
если файл остается пустым

Заполнение файла settings:
1 строка - выбор рефлектора: одна цифра - (UKW-A - 1, UKW-B - 2, UKW-C - 3)
2 строка - используемые роторы: 3 неповторяющиеся цифры от 1 до 5 через пробел
3 строка - начальные положения роторов: 3 любые цифры от 1 до 26 через пробел
4 строка - ввод коммутаций: если их нет то написать 0, если есть писать в формате
A B;C D;E F (! только английские буквы )
в коммутациях может быть связь только между двумя буквами
Если будут ошибки в заполнении машина не начнет работу

2) Принцип работы
Программа разбивает сообщение на символы. Каждая буква проходит шифрование в следующем
порядке: коммутация -> входной диск(ETW) -> правый ротор(тот который введет третьим) ->
-> средний ротор -> левый ротор -> рефлектор(UKW) -> левый ротор -> средний ротор -> правый ротор ->
-> входной диск -> коммутация -> итоговая буква

Таким образом получается простая цепь, которая работает в двух направлениях
